## Опис проєкту

У межах лабораторної роботи створюється персональний портфоліо-сайт майбутнього фронтенд-розробника.

Сайт складається з таких сторінок:

1. `index.html` — головна сторінка.
2. `about.html` — сторінка з інформацією про автора.
3. `contacts.html` — контактна сторінка з формою зворотного зв'язку.
